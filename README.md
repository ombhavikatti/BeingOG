<!--                       WHAT IS BEINGOG?                             -->

<div align="center">
## <img src="https://media.giphy.com/media/iY8CRBdQXODJSCERIr/giphy.gif" width="30" />&nbsp; What is BeingOG?
</div>
<br />
<div align="center">
<table>
<tr>
<td>
<br />
&nbsp;&nbsp;&nbsp;&nbsp;**BeingOG** is a **premium productivity ecosystem** — built for engineering students,&nbsp;&nbsp;&nbsp;&nbsp;<br />
&nbsp;&nbsp;&nbsp;&nbsp;competitive-exam aspirants, and placement-focused learners who want more than a to-do app.
&nbsp;&nbsp;&nbsp;&nbsp;It combines **habit tracking**, **study analytics**, **real-time social accountability**,<br />
&nbsp;&nbsp;&nbsp;&nbsp;**AI coaching**, and **placement readiness** into a single, beautiful, gamified platform.
<br />
</td>
</tr>
</table>
</div>
<br />
<table>
<tr>
<td width="50%" valign="top">
### &nbsp;&nbsp;🔴 &nbsp; The Problem
<br />
> Students struggle to build consistency. They lack accountability, lose motivation, waste hours, and have no way to measure whether they're actually placement-ready. Existing apps solve **one** of these problems. None solve **all** of them.
</td>
<td width="50%" valign="top">
### &nbsp;&nbsp;🟢 &nbsp; The Solution
<br />
> BeingOG brings **productivity**, **social accountability**, and **AI-driven coaching** into one polished, gamified experience — so students stop drifting and start compounding.
</td>
</tr>
</table>
<br />

```
BeingOG/
│
├── 📦 apps/                          # All runnable applications
│   ├── 🖥️  web/                      # Next.js frontend        → Vercel
│   ├── ⚙️  api/                      # NestJS backend          → Railway

│
├── 📚 packages/                      # Shared code between apps
│   ├── 🎨 ui/                        # Reusable React components
│   ├── 🔷 types/                     # Shared TypeScript types
│   ├── ⚙️  config/                   # Shared ESLint / TS / Tailwind configs

│
├── 📖 docs/                          # Architecture & feature docs
│
├── 🔄 .github/
│   └── workflows/                    # CI/CD pipelines
│
├── .gitignore
├── pnpm-workspace.yaml               # Monorepo definition
├── turbo.json                         # Turborepo config
└── README.md
```
